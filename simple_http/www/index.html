About This Server

This is a custom HTTP/1.1 server implementation built from scratch using raw TCP sockets in Python. It demonstrates fundamental concepts of network programming and the HTTP protocol.

Supported Features

  • GET - Retrieve files from the server
  • POST - Submit data to /post endpoint
  • PUT - Upload files to the server
  • HEAD - Get headers without body content

Server Information

Port: 8080

Protocol: HTTP/1.1

Transport: TCP

Status: ● Online
