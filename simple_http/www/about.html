Project Description

This assignment implements a basic HTTP/1.1 server using Python with raw TCP sockets. The server handles HTTP requests, parses headers and bodies, maps paths to files, and generates proper HTTP responses.

Supported Methods

GET - Retrieve files from server (returns 200 OK or 404 Not Found)

POST - Submit data to /post endpoint (appends to post_data.txt)

PUT - Upload files to server (returns 201 Created or 200 OK)

HEAD - Get headers only without body content

Technical Details

Port: 8080, Protocol: HTTP/1.1 over TCP, Socket Type: AF_INET, SOCK_STREAM
